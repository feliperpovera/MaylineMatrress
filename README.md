# Maylin Mattress

Static marketing site and lead capture portal for `Maylin Mattress`.

## Local run

Requirements:
- `Node.js 20+`

Commands:

```bash
npm install
npm start
```

Website:
- `http://127.0.0.1:3000`

Portal:
- `http://127.0.0.1:3000/portal`

## Current delivery scope

This version includes:
- Homepage copy updated from `Miami` to `South Florida` where requested.
- Hero video end-logo reveal.
- Products reordered with `Dual-Sided Pillow Tops` in third position and `Latex Mattress` in fourth.
- Expanded carpentry messaging.
- Reviews section with BusinessRate award visual.
- About section updated with new image treatment and revised factory visit block.

## Start script
